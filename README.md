# Gold 18k Deal Finder — Cash Converters Scraper + €/gram Dataset

This project scrapes 18k gold jewelry listings from Cash Converters Spain (Canary Islands) and generates a dataset to quickly identify potential “best value” items based on **€/gram**.

## Features
- Scrapes listing + product pages
- Extracts: price, grams, stone type, shop code, store info
- Computes **€/gram**
- Exports datasets to CSV
